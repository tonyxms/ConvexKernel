###
header.R: Include different R packages and R files we need to use.

Michal_6.R: Michalewicz function with six active dimensions

Michalewicz.R: Michalewicz function with two active dimensions

optkernel_method_3.R: Optimal kernel method (the proposed method in the article) without leave-one-out Cross Validation

optkernel_method_loocv.R: Optimal kernel method with leave-one-out Cross Validation

optkernel_pred.R: The function computes the predicted responses


Eg_Michal_boxplot.R: Example for Michalewicz function (section 6.1)

Eg_Borehole_boxplot.R: Example for Borehole function (section 6.2) 

Eg_satellite.R: Example for predicting satellite drag (section 6.3); train data: CD_GRACE_1000; test data: CD_GRACE_100

Eg_Michalewicz_function.Rmd: Simple example for different methods of 2-dimensonal Michalewicz function (get the true & predicted surface of Michalewicz function)

test_cross_validation.Rmd: Plot for the Leave-one-out Cross-Validation error
